# fortran-cli-make-hello-world

## Description
This is a POC project for fortran.
Uses make file with  gfortran compiler.

## Tech stack
- fortran
- gfortran
- make

## Docker stack
- ubuntu:18.04

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
